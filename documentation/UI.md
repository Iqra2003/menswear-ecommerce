# -------------------SPARE PART MATCHING---------------------



1. ## Introduction:





The Spare Parts Matching System is a web application designed to help retailers and wholesalers manage vehicle parts inventory, track stock levels, add new parts, and perform image-based matching.

The system is especially useful for business workflows like spare parts business — managing stock, sending quotations, and quickly identifying parts.



2\. Application Features:





* **Dashboard**



Displays a quick overview of:



Total Parts



Low Stock Parts



Recent Searches



Also includes:



Search bar



List of recently added parts





* **Add Spare Part Page**



Form for adding new spare parts into the inventory system.



Includes fields:



Part Name



Part Number



Stock Quantity



Upload Part Image



&nbsp;	Preview the uploaded image



&nbsp;	Save or Delete option



*Purpose:*



Add new items to the inventory



Upload images that support image-matching functionality





* **Parts List Page**



Displays all parts stored in the database with pagination.



*Includes:*



Search bar to search by name/number



Filter dropdown (e.g., All, Low Stock, Category)



Table with columns:



Part Name



Part Number



Stock



Actions → Edit button



Pagination (1, 2, 3…)



*Purpose:*



Quickly browse and manage inventory



Edit existing parts



Check which parts are low on stock



* **Image Matching Page**



A page where we will upload a spare part photo to check if it is in stocks



*Purpose:*



Upload a spare part photo



System matches it with the closest item in the database



Useful for:



Customers sending random images of parts



Faster identification
